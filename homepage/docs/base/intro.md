# 认识社区

mcpp是一个现代C++爱好者社区

## MVV - 使命、愿景、价值观

- **`使命`**: 构建模块化的现代C++生态, 并加速开源文化和现代C++文化的传播
- **`愿景`**: 成为有影响力的现代C++爱好者社区
- **`价值观`**: 开源共创、真诚热爱、开放包容

## 具体要做的事情

- 技术&文化传播 - 现代C++影响力、模块化特性、教程
- [mcpplibs - 现代C++模块化库集合](https://github.com/mcpplibs)
- [社区文化 - 爱好者聚集地、技术交流、社区活动]( https://mcpp.d2learn.org/forum)
